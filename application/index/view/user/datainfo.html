第一卷 人世沧桑 第一章 沙谷记事

“嘎...”

随着干裂门栓涩然的摩擦声，一扇擦拭干净,但已是十分老旧的木门被徐徐推开。

从中走出个十八九岁样貌平凡的少年，少年中等身材，但脸色上血色不多，显得有些苍白，其神色更是有些木讷。

一阵晨风拂过，打着几个补丁的灰色长衫随风摆动，紧贴在身躯上露出其下骨多肉少的干瘦身形。

此时，天色刚刚发白，四周静寂无声，没有任何人影。

少年本还木讷的脸庞上，一双呆滞的眼眸里忽然露出了几分神采，他左右打量了一圈后，撩起下摆束紧在腰带上，顺着泥土夯实的小道向远处一座山坡缓步跑去。

短短一刻钟的时间，长衫背部已被湿透，豆大的汗珠顺着额头及两鬓处不断滑下，滴落在脚下带起的尘土上。

瞅着还在远处的山顶，少年咬了咬牙，使自己急促的呼吸声尽量平稳些，继续挪动双脚向前跑去，虽说速度比起刚开始已慢了很多，却没有丝毫停下来的意思，只是步伐变得更为凌乱蹒跚，显得十分狼狈。

通往坡顶的山间小道蜿蜒而曲折，其上布满了细小碎石，很不好跑。

两侧是高大繁茂的树木，枝叶间相互交集缠绕，好似一把翠绿的大伞盖住了整片天空。此时的晨光好似缕缕金丝般，透过枝丫细缝洒落下来，把这幽静的小道映得斑斓不定，平添了几分色彩。

空气格外的清新，少年不由再次深吸了一口，并在双手帮助下，狼狈登上了山坡顶部。

这种好似晨跑一般的行为，算是近一年来，这少年给自己定下的锻炼任务，虽不激烈，可这幅羸弱身躯以及身体上那极为陌生的生涩感，还是让他透出了几分无奈之色。

抹去额头上的汗珠，一股山风忽然吹过，早已湿透并披于后背的乌黑长发被高高荡起。

就见这少年的身躯不禁微颤，本还疲惫的神情竟微微振奋，一股久违的舒适感不断从身体各处传来，这种感觉让他十分迷醉，每天也只有到了这个时候，这幅羸弱身躯才好似真正的属于他自己。

当少年从沉醉中清醒过来，踱步踏上块巨大而平整的石块向山坡下望去，那里坐落着一片低矮而简陋的房舍，稀稀落落依山而建，道道炊烟正从那片房舍中袅袅升起，没入苍穹之中，远远望去竟透出了几分世外桃源之味。

“哎....”

望着这片宜人的田园风光，少年的眉梢没有随之舒展，而是一声轻叹从口中传出，本还木讷的脸颊上也逐渐复杂了起来。

山下这片小小村落，名为沙谷村，算是个极为偏僻并且资源贫乏的小村庄。它位于苍元国东南方一处角落里，在这四周众多的村落中，也是属于那种很不起眼的存在。

而这少年也并不是村落之人，准确的说，是这幅身躯的神魂早已换了主人。

林星宇是他本来的名字，自从来到这里已是二年有余，现在想起这段离奇的经历，还是如坠梦中没有丝毫的真切感。

两张熟悉而温馨的面容又悄悄浮现在自己的脑海中，林星宇不由的阖上眼睑，底叹了几声，脸颊上的神情显得更为落寂了些。

好一会过后，他才摇了摇头，又渐渐恢复了那副木讷神情。

这座小山坡虽然不高，可远处那条蜿蜒巍峨的山脉还是依稀可见，听村中的老人说，这条山脉的边缘处距离村头东面只有不到百余里，被称为苍元山脉。

连绵起伏的苍元山脉足足有数万里之长，宛如一条黑色巨龙静静盘卧在这片大地上，远远望去，幽深而谧静。

山脉中栖息着众多凶兽以及那传闻中的妖兽，其中不光凶险异常，同时也孕育着许许多多就连修行者也梦寐以求的珍稀药材。

由于沙谷村地处偏僻，又距离苍元山脉较近，这里时常会发生些凶兽甚至妖兽袭击的事件，使得村中人口一直无法兴旺，总共只有三十来户的村民，世代过着男耕女织的生活，朴实而平淡。

初夏，天气说变就变，刚刚还是碧云连天，转眼已是暴雨将临。

黑压压的云层一片接着一片，天空以肉眼可见的速度开始变暗，沉闷的让人喘不过气来，远处幽静的山林被平地而起的狂风刮的左摇右摆，枝叶上不断发出萧萧飒飒的呜鸣声，像是在嘀嘀哭泣，又好似无数怪物正在林中奔走，使得气氛显得更为阴沉。

早已吹干的灰色长衫，发出了猎猎作响声，少年背负着双手，抬头向远处的天空凝神望去。

只见那片漆黑厚实的乌云在劲风搅动下，正上下剧烈翻滚，好似有着什么妖物正在其中追逐嬉戏，使空气中本就沉闷的气息变得更为压抑。

其实林星宇那张嫩稚的面容本来还算清秀，只是一双眼睛明显比别人要小上一圈，这才变得格外普通，不过脸颊上的鼻梁却是笔直而高挺，最前端还稍稍向内弯勾，使那普通的面容上又透出几分冷冽之气。

收回目光，他低下头向坡脚处望去，依稀能看那里有着几亩稀稀落落的田地，低不可闻的嘀咕声悄然传出：

“这儿的气候还真是变化无常，只希望这场暴雨不会影响灵谷产量，否则计划又得改动。”

看着远处天边好似已开始飘起了细雨，林星宇只得一边摇着头，一边向山下踱步走去，只是那木讷的面容上，无奈的神情始终没有消散。

这两年来，这幅身躯虽说已经换了神魂，可模样并没有丝毫改变，为了不被村民察觉出异样，他只得装出一副呆傻模样，好在他本就是个成年人，这才花了两年时间勉强学会了这里的语言文字。

真正让林星宇头疼的还是这幅羸弱身躯，不管是在走路，还是田间农活，都无时无刻不传来一种极为生涩的异样感，就好似这幅身躯并没有完全接纳他的神魂一般。

不过，经过这一年来不停的锻炼，这种感觉似有所好转，可行动间还是透着几分别扭。

他本想多了解些这里情况，再去寻找解决办法，可这沙谷村实在是太过偏僻，消息闭塞不说，就连通往其他城镇的山道也同样危险无比，而他对于这方世界的了解，也只停留在村中老人们的闲谈中。

通过这些只言片语，林星宇推测出，此地极有可能是个能够通过修行而得道长生的异世界，只是村中并没有任何修行之人，对此，他还是半信半疑。

沙谷村虽然偏僻穷困，可其中村民都是些老实本分之人，这附近最为危险的还属那些时常出没的野兽，这些野兽不光是体形庞大，而且嗜血凶残，村民称之为凶兽。

为了保护村民安全，每几个村庄里，便会有一到两个被称为武者的人在此驻扎巡视，他们不光负责抵御凶兽，必要时还要协调村民撤离前往他处躲避。

虽说这几个武者平时都在各村里游走不定，可这两年下来，在几次与凶兽的激烈战斗中，林星宇还是注意到，这些武者都拥有着极为强大的力量，简直就像前世的电视画面中那些武林高手。

林星宇所见的这几个武者，体形健硕，孔武有力，激战时更能跃起一丈来高，对于寻常人，不说能以一敌百，一人打上几十个还是没有丝毫问题。

可即便如此，在他看来照样没有脱离人类范畴，如果真有村民口中那种修道长生之人，应绝不止这区区手段，对此，林星宇还是很确信自己判断的。
